# Amazon_Vine_Analysis

## Overview
> During this analysis we used the ETL process to analyze Amazon reviews for video games. To perform the ETL process we used the PySpark from Google Colab.

### Tools 
  * AWS, RDS, S3
  * Google Colab
  * Pyspark
  * SQL, pgAdmin
